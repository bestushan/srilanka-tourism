# Sri Lanka Tourism Web App

A cinematic, minimalist tourism website showcasing Sri Lanka's destinations with premium design and immersive scroll-based storytelling.

## Features

- **Cinematic Design**: Full-viewport hero sections with grain textures and parallax scrolling
- **Golden Accents**: Strategic use of golden yellow (`#FFD700`) highlighting CTAs and key interactions
- **Responsive Layout**: Mobile-first design with breakpoints at 640px, 1024px, and 1280px
- **Smooth Animations**: Framer Motion-powered entrance effects, hover states, and scroll interactions
- **Premium Typography**: Playfair Display serif headings paired with Inter sans-serif body text
- **Dark Aesthetic**: Deep charcoal backgrounds (`#0A0A0A`) creating luxury and visual depth

## Tech Stack

- **Fonts**: Playfair Display & Inter (Google Fonts)
- **Animations**: Framer Motion
- **Images**: Unsplash & AI-generated imagery
- **Design System**: CSS variables with 8px spacing grid

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start dev server: `npm run dev`
4. Build for production: `npm run build`

## Design Tokens

- **Colors**: Deep Charcoal (`#0A0A0A`), Pure White (`#FFFFFF`), Golden Yellow (`#FFD700`), Muted Gray (`#A0A0A0`)
- **Spacing**: Multiples of 8px (8, 16, 24, 32, 48, 64)
- **Transitions**: 0.3s ease-in-out default
